# How to Install Tailwind CSS v4 (Step-by-Step Guide)

### Step 1: Prerequisites

Before installing Tailwind CSS v4, make sure you have the following installed on your machine:

- Node.js (v16 or above)
- npm (comes with Node.js) or yarn
- A code editor like VS Code

> Tip: Check your Node.js and npm versions by running:


### Step 3: Install Tailwind CSS

Install Tailwind CSS v4 via npm (or yarn). Run:

```bash
npm install -D tailwindcss@latest postcss autoprefixer
```

Then, generate the Tailwind configuration files:

```bash
npx tailwindcss init -p
```

This will create two files in your project root:

- `tailwind.config.js` â€” Tailwind configuration
- `postcss.config.js` â€” PostCSS configuration

### Step 4: Configure Your Template Paths

Open `tailwind.config.js` and specify the files Tailwind should scan for class names. For example:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{html,js,ts,jsx,tsx}", // Scan all HTML and JS/TS files
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

> Tip: The `content` array is crucial for purging unused CSS in production.

### Step 5: Add Tailwind Directives to Your CSS

Create a CSS file (e.g., src/styles/tailwind.css) and add the following:

```bash

@tailwind components;
@tailwind utilities;
```

- These directives import Tailwindâ€™s base styles, components, and utility classes.

### Step 6: Include Tailwind in Your Project

- <b>For HTML projects:</b>
Include your compiled CSS file in the `<head>` of your index.html:

```html

```

- <b>For React/Next.js projects:</b>
  Import the CSS file in your main entry point, e.g., src/index.js or app/layout.js:

```js
import "./styles/tailwind.css";
```

### Step 7: Build Your Tailwind CSS

You can build Tailwind CSS using the CLI:

```bash
npx tailwindcss -i ./src/styles/tailwind.css -o ./dist/output.css --watch
```
